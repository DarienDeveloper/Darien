¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. Permite separar el contenido de un documento (su estructura) de su diseño (colores, fuentes, márgenes, etc.), facilitando la gestión del estilo y la apariencia de las páginas web.

¿Cuál es la función principal de CSS?


La función principal de CSS es controlar la presentación visual de los elementos en una página web. Esto incluye aspectos como el diseño, los colores, las fuentes y el espaciado. CSS permite a los desarrolladores web aplicar estilos de manera consistente en múltiples páginas y adaptar el diseño a diferentes dispositivos.

¿Cómo se puede incluir CSS en un documento HTML?

Hay tres maneras principales de incluir CSS en un documento HTML: 

            
            <h1 style="color: blue;">Hola Mundo </h1>
            
            
                
                <head> <style="color: blue;">Hola Mundo </h1>
                
                
                    
                    <head>
<style>
h1 { color:blue }
                    </style> </head>

¿Qué son los selectores en CSS?

  1. Selectores de tipo: Seleccionan todos los elementos de un tipo específico (por ejemplo, p para párrafos).
  2. Selectores de clase: Seleccionan elementos que tienen una clase específica (por ejemplo, .clase).
  3. Selectores de ID: Seleccionan un elemento único con un ID específico (por ejemplo, #id).
  4. Selectores de atributo: Seleccionan elementos con un atributo específico (por ejemplo, [type="text"]).

¿Qué son las propiedades y valores en CSS?

Las propiedades son características de los elementos HTML que se pueden modificar mediante CSS. Cada propiedad tiene un conjunto de valores posibles que determinan cómo se aplica el estilo. La sintaxis general es: 

    
       selector { propiedad: valor; }

¿Cómo se aplica el modelo de caja en CSS?

El modelo de caja (box model) es un concepto fundamental en CSS que describe cómo se estructuran los elementos en una página web. Cada elemento se representa como una caja rectangular, que consta de:

  1. Contenido: El texto o imágenes dentro del elemento.
  2. Padding (relleno): Espacio entre el contenido y el borde de la caja.
  3. Border (borde): Línea que rodea el padding y el contenido.
  4. Margin (margen): Espacio exterior que separa el elemento de otros elementos.

¿Qué es el "cascading" en CSS?

El "cascading" se refiere a la forma en que CSS aplica reglas de estilo a un documento. Cuando hay múltiples reglas que pueden aplicarse a un mismo elemento, el navegador determina cuál utilizar según:

  1. Especificidad: Cuanto más específico sea el selector, mayor prioridad tendrá.
  2. Orden: Si dos reglas tienen la misma especificidad, se aplicará la última definida.
  3. Importancia: Las reglas con !important tienen prioridad sobre las demás.

¿Qué son las media queries?

Las media queries son una característica de CSS que permite aplicar estilos específicos basados en ciertas condiciones, como el tamaño de la pantalla o la orientación del dispositivo. Esto es clave para el diseño responsivo. Un ejemplo básico es: 

    
    @media(max-width: 600px) 
body  { background-color: lighblue; }}

¿Cuáles son algunos estilos comunes que se pueden aplicar?

Algunos estilos comunes en CSS incluyen:

  1. Color y fondo: color, background-color, background-image
  2. Tipografía: font-family, font-size, font-weight, text-align
  3. Espaciado: margin, padding
  5. Dimensiones: width, height, max-width, max-height
  6. Bordes: border, border-radius
    7. Display: display, position, float

¿Cómo se puede usar CSS para crear un diseño responsivo?

Para crear un diseño responsivo, puedes utilizar:

  1. Media Queries: Para aplicar estilos diferentes según el tamaño de la pantalla.
  2. Unidades relativas: Usar unidades como %, em, y rem en lugar de unidades fijas (px) para que los elementos se ajusten según el tamaño del contenedor.
  3. Flexbox y Grid: Utilizar modelos de diseño como Flexbox y CSS Grid para crear estructuras que se adapten a diferentes tamaños de pantalla.
  4. Imágenes fluidas: Hacer que las imágenes sean escalables usando max-width: 100%; para que no excedan el ancho de su contenedor.
